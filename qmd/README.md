# Quarto Bestanden voor DigComp 3.0 Vertaling

Deze map bevat de Quarto (`.qmd`) bestanden die gebruikt worden voor de uiteindelijke vertaling in het Nederlands van het DigComp 3.0 document.

## Inhoud
- `DigComp3.0-Nederlands.qmd`: Het hoofdbestand met de Quarto broncode.
- `apa.csl`: Citatiestijl bestand.
- `references.bib`: Bibliografie bestand.
- `DigComp3.0-Nederlands.docx`: De gegenereerde Word-output.
- `custom-reference.docx`: Referentiebestand voor de Word-styling.
