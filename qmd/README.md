# Quarto Files for DigComp 3.0 Translation

This directory contains the Quarto (`.qmd`) files used for the final Dutch translation of the DigComp 3.0 document.

## Contents
- `DigComp3.0-Nederlands.qmd`: The main file with the Quarto source code.
- `apa.csl`: Citation style file.
- `references.bib`: Bibliography file.
- `DigComp3.0-Nederlands.docx`: The generated Word output.
- `custom-reference.docx`: Reference file for Word styling.
